# Obter uma Assinatura Azure

Caso você não tenha uma Assinatura Azure.

## Avaliação Gratuita do Azure

Você pode obter uma assinatura de $200 se [se inscrever no Azure](https://azure.microsoft.com/en-us/pricing/purchase-options/azure-account).


## Azure Classroom

::: tip NOTA
Esta oferta se aplica apenas a eventos fornecidos pela Global AI Community em eventos como AgentCon.
:::

Você pode seguir as instruções abaixo e obter uma Assinatura Azure com $10 de créditos válidos por 5 dias.

### Passo 1 - Solicitar assinatura
- Navegue até: [https://aka.ms/JoinEduLab](https://aka.ms/JoinEduLab)
- Faça login com uma conta Microsoft ou GitHub

![Captura de tela do cadastro Azure Classroom](/public/azure-classroom/001.png)

- Digite <!--@include: ./variables/azure-code.md--> o `código de convite` e `clique em resgatar`
- Aguarde o email abaixo chegar

### Passo 2 - Ativar assinatura
- Procure pelo email abaixo chegando na sua caixa de entrada.
![Você tem uma solicitação para assumir a propriedade de uma assinatura Azure](/public/azure-classroom/002.png)
- Clique no botão `Accept Ownership`
- Faça login com a mesma conta do passo 1

![](/public/azure-classroom/003.png)
- Após o login, clique em `Accept` para entrar no tenant Open Hack.

![](/public/azure-classroom/004.png)
- Você pode usar o nome de assinatura sugerido ou criar o seu próprio.
- Clique em `Review + Accept`

![](/public/azure-classroom/005.png)
- Clique em `Accept`
- Sua assinatura está sendo criada

### Passo 3 - Bom trabalho
- Pode levar alguns minutos para criar sua assinatura.
![](/public/azure-classroom/007.png)

- Quando sua assinatura estiver pronta, você receberá um email com o assunto: **Your Azure subscription is ready**
![](/public/azure-classroom/006.png)

- Você também pode encontrar o status do processo de criação da assinatura em notificações.
- Clique em "Go to subscription" na notificação ou `Go to the Azure portal` no email

Bom trabalho!

### Dicas e Truques

#### Não consegue encontrar sua assinatura?
- Sua assinatura é criada no Tenant/Diretório: "Global AI Community"
- Às vezes você precisa trocar de tenants antes de ver a assinatura. [Clique aqui](https://portal.azure.com/#settings/directory) para trocar de Diretório.
- Às vezes você precisa ajustar seu [filtro de Assinatura](https://portal.azure.com/#view/Microsoft_Azure_Billing/SubscriptionsBladeV2) para a assinatura aparecer.
